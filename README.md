# Construction Labour Recruitment MVP

A simple local web app for matching available construction workers to contractor job requests.

## Run locally

Open `index.html` in a browser. No install or build step is required.

## Features

- Add worker profiles with trade, tickets, availability, and reliability score.
- Add contractor job requests with trade, location, start date/time, and optional duration.
- Match available workers by exact trade and rank them by reliability score.
- Manually assign any worker to any job from the admin view.
- Update reliability when a worker completes on time, is late, or no-shows.
- Data is saved in browser localStorage.
